--- COMPILACIÓN BITQUEST ---
Para la correcta compilación del proyecto se hace necesario hacer uso del archivo "Compilar.bat".
Se debe ejecutar dentro de una carpeta con el código fuente completo del proyecto.
Después de ejecutarse se cierra la ventana y se tendrá que generar un archivo ejecutable, este es el programa.

PD: El proyecto aunque pueda tener parte de compatibilidad con LINUX, no es posible correrlo en LINUX ni en MACOS, por lo que el proyecto únicamente corre en Windows
